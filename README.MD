# input


É um prompt avançado para euphoria.


## Funcionalidades

- Mascara de password
- Limitador de caracteres
- Texto de prompt
- Mudança de cores do texto e do cursor.


## Uso/Exemplos

```euphoria
include input.e

input("Write here: ")
input("write password here(max 4 digits): ", TYPE_PASSWORD, 4) -- mask * with max of 4 char
```
## Stack utilizada

**Lang:** Euphoria
**OS:** windows, linux, macos
